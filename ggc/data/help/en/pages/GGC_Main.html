Help is not available yet. Framework is created and implemented, but we still need to write all help pages. This is planned for version 0.4.

Quick How-To:
1. Go to "Tools\Preferences" and set all preferences that you think that you need to change.
2. Use BGs\Daily to add daily values
3. If you set "Printing" in Preferences you can also print everything...


Everything else is still in development... You can use graphs, but most of them still have some problems.


Last added functionality is Foods... here you have three types of databases for foods and meals... This is fully functional... You can add foods in User Database and create meals in Meals database... Data created here can then be used to "import" into Daily Values and CH is extracted from it (for Carb Counting)
